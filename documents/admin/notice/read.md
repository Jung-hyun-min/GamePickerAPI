# Get all notice

**Method** : `POST`

**URL** : `/admin/notices`

**Auth required** : `False`

**Data constraints** 
```
NULL
```

**Code** : `200 OK`

**Content example**
```
{
    notices: [
        {
            id: 1,
            title: "First notice",
            value: "My first notice",
            created_at: "2018-10-04T12:39:39.000Z"
        }
    ]
}
```
